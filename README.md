# PPO_labs

### Название проекта: **Сайт для танцевальной студии**

## **Краткое описание идеи проекта:**

Web-приложение, на котором будет располагаться информация для клиентов танцевальной студии. 
Основная идея заключается в том, что клиент сможет бронировать залы, записываться на классы/курсы онлайн, авторизироваться в личном кабиенте, в котором отображаются посещения, расписание выбранного филиала, действующий абонемент. Клиенты также могут оставлять отзывы с рейтингом, просматривать информацию: цены на абонементы, расписание, контактную информацию и так далее.

## **Краткое описание предметной области**

Предментная область включает в себя следующие сущности:

1. Танцевальный зал.
2. Расписание.
3. Абонемент.
4. Клиент(Пользователь).
5. Студия.
6. Адрес.
7. Типы абонементов.
8. Открытые классы.
9. Специальные курсы.
10. Класс с приглашенным педагогом.
11. Бронирование.

5. Отзывы (хранение оставленных комментариев о комнате/зоне в антикафе)
6. Меню (Блюда).
7. Пакеты.

Роли:

1. Пользователь.
2. Клиент.
3. Администратор.

Краткое описание предметной области: 
Существует четыре основных действия, реализуемых в рамках танцевальной студии: 1) Запись на открытые классы с фиксированный расписанием в течение недели; 2) Запись на одноразовые классы приглашенных педагогов; 3) Бронирование свободных залов; 4) Запись на специальные курсы от педагога (чаще всего с темой прокачки определенных навыков);
Также сохраняются посещения уже прошедших классов потребителями -> возможно дальнейшая аналитика на основе этих данных.
